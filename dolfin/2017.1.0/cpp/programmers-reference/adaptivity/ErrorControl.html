class ErrorControl¶

Parent class(es)

  • Hierarchical
  • Variable

(Goal-oriented) Error Control class. The notation used here follows the notation in “Automated goal-oriented error control I: stationary variational problems”, ME Rognes and A Logg, 2010-2011.
