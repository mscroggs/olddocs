class Function¶

Parent class(es)

  • GenericFunction
  • Hierarchical

This class represents a function \(u_h\) in a finite element function space \(V_h\), given by

\[u_h = \sum_{i=1}^{n} U_i \phi_i\]

where \(\{\phi_i\}_{i=1}^{n}\) is a basis for \(V_h\), and \(U\) is a vector of expansion coefficients for \(u_h\).
