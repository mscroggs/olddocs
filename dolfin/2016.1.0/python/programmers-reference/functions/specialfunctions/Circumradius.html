Circumradius¶

dolfin.functions.specialfunctions.Circumradius(mesh)¶

Return symbolic cell circumradius for given mesh.

Arguments
mesh
a Mesh.

Example of usage

mesh = UnitSquare(4,4)
vol = Circumradius(mesh)
