class ImplicitSurface¶

This class is used to define a surface via a function f(x) -> R, where for a point y on the surface f(y) = 0. WARNING: This class is experimental and likely to change.

ImplicitSurface(Sphere s, std::string type)¶

Create an isosurface

Arguments
s (Sphere)
Bounding sphere for surface.
type (std::string)
Isosurface type. One of “manifold”, “manifold_with_boundary” or “non_manifold”.
Example
IsoSurface isosurface(Sphere(Point(0.0, 0.2, 0.4), 1.0),
                      "manifold_with_boundary");
double operator()(const Point &point) const = 0¶

Return value of isosurfacce function. This function is overloaed by the user. For closed volumes, the function should evaluate to < 0 on the interior.

Arguments
point (Point)
The point at which to evaluate the isosurface function. or “non_manifold”.
Returns
double
Isosurface function value.
double f0(const Point &point) const¶

Signed distance function surface. If f0(p) = 0, the point p is possibly on the surface, which case ImplicitSurface::f1 can be called to check.

double f1(const Point &point) const¶

For a point for which f0 approx 0, return <= is point is on is on the surface. Can be used for creating open surfaces by discarding with any artificial closure.
